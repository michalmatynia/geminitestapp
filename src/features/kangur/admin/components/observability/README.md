# Kangur Admin Observability Components

This folder owns observability-specific admin UI building blocks used by the
Kangur AI tutor and knowledge-graph monitoring workspace.

## Layout

- `AiTutorBridgeMetrics.tsx`: AI tutor bridge metric cards
- `GuideEditorSections.tsx`: guide editor section composition
- `KangurAiTutorNativeGuideEntryEditorContext.tsx`: guide-editor context
- `KnowledgeGraph*.tsx`: knowledge-graph status and query preview sections
- `LogSections.tsx`: log output sections
- `MetricCards.tsx`: shared observability metric cards
- `SummaryContent.tsx`: observability summary composition
- `guide-editor-utils.ts`, `helpers.tsx`, `utils.ts`: non-page observability helpers
