# Legacy Milkbar Designers FastComet Folder

Milkbar CMS uploads now use the `uploads.milkbardesigners.com` storage origin.
The copy-ready upload endpoint scaffold lives at:

```text
hosting/fastcomet/uploads.milkbardesigners.com/public_html/
```

This legacy folder is kept because it may contain local mirror artifacts from
the previous `milkbardesigners.com` FastComet layout.
