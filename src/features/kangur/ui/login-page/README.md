# Kangur UI Login Page

This folder owns Kangur login/signup page support such as forms, constants,
Turnstile integration, and login-flow logic.

## Layout

- `login-constants.ts`: login-page constants and Turnstile script metadata
- `login-context.tsx`: login flow context
- `login-forms.tsx`: login form components
- `signup-forms.tsx`: signup form components
- `use-login-logic.ts`: login flow orchestration helpers
- `use-turnstile.ts`: Turnstile script/bootstrap hook
- `__tests__/`: login-page tests for this folder
