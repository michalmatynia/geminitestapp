# Temporary TODO Backlogs

This directory contains short-lived debt queues and historical TODO notes. It is
useful for work-in-progress triage, but it is not the canonical home for stable
architecture, policy, or feature documentation.

## Current Files

- [`TODO_ESLINT_FIXES.md`](./TODO_ESLINT_FIXES.md)
- [`TODO_ESLINT_FIXES_PHASE5.md`](./TODO_ESLINT_FIXES_PHASE5.md)
- [`TODO_ESLINT_FIXES_PHASE6.md`](./TODO_ESLINT_FIXES_PHASE6.md)
- [`TODO_FIX_TESTS.md`](./TODO_FIX_TESTS.md)

## Placement Rule

- Keep this directory for temporary queue-style notes only.
- When a TODO turns into scheduled cross-feature work, promote it into
  [`docs/plans/`](../plans/README.md) or the relevant feature folder.
- When a TODO becomes durable operational guidance, rewrite it into a canonical
  runbook or platform doc and leave this folder as temporary backlog only.
