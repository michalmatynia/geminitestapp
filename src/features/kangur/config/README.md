# Kangur Config

This folder owns Kangur route config, launch-route behavior, and access rules.

## Layout

- `launch-route.ts`: launch-route defaults and selection helpers
- `page-access.ts`: role-aware page and slug access rules
- `pages.ts`: Kangur page inventory and lazy page registration
- `routing.ts`: public/internal route construction and parsing helpers
- `__tests__/`: config tests that are not owned by a narrower nested folder
