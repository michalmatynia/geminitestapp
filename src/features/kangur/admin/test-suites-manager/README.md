# Kangur Admin Test Suites Manager

This folder owns the admin workspace used to browse, edit, and organize Kangur
test suites and question groups.

## Layout

- `QuestionsManagerWorkspace.tsx`: top-level workspace shell
- `GroupMetadataPanel.tsx`: group metadata editing panel
- `LibraryStatusSidebar.tsx`: suite/library status sidebar
- `TestSuiteLibraryHeader.tsx`: test-suite library header UI
- `TestSuitesModals.tsx`: modal composition for the workspace
- `test-suites-manager.context.tsx`: workspace context
- `test-suites-manager.contracts.ts`: workspace-facing types/contracts
- `test-suites-manager.logic.ts`: non-visual workspace logic helpers
