# Kangur Lesson Template Repository

This folder owns the Kangur lesson-template repository contract, template
normalization, and MongoDB-backed localized template implementation.

## Layout

- `index.ts`: repository entrypoint
- `mongo-kangur-lesson-template-repository.ts`: MongoDB bootstrap and template query implementation
- `normalize-kangur-lesson-template.ts`: template normalization helpers
- `types.ts`: repository-facing contract types
- `__tests__/`: repository tests for this folder
