# Kangur UI Routing

This folder owns Kangur UI-side route access, managed path helpers, pending
route loading snapshots, and transition skeleton selection.

## Layout

- `managed-paths.ts`: managed Kangur href and alias-path helpers
- `pending-route-loading-snapshot.ts`: pending-route loading snapshot sanitization
- `route-transition-skeletons.ts`: route-to-skeleton mapping helpers
- `useKangurRouteAccess.ts`: route-access hook for UI routing decisions
- `__tests__/`: routing tests for this folder
