# Kangur Admin Hooks

This folder owns admin-only hooks for content mutation flows, starter recipes,
block-list drag-and-drop, and quick insert actions.

## Layout

- `useBlockListDnd.ts`: block-list drag-and-drop behavior
- `useKangurLessonMutations.ts`: lesson mutation helpers
- `useKangurPageContentMutations.ts`: page-content mutation helpers
- `useKangurQuestionsMutations.ts`: question mutation helpers
- `useKangurSettings.ts`: admin settings query helpers
- `useKangurStarterRecipes.ts`: starter recipe helpers
- `useQuickInsertActions.ts`: quick insert action helpers
