# Kangur Lesson Section Repository

This folder owns the Kangur lesson-section repository contract, section
normalization, and MongoDB implementation.

## Layout

- `index.ts`: repository entrypoint
- `mongo-kangur-lesson-section-repository.ts`: MongoDB bootstrap and section query implementation
- `normalize-kangur-lesson-section.ts`: section normalization helpers
- `types.ts`: repository-facing contract types
- `__tests__/`: repository tests for this folder
