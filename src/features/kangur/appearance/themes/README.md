# Kangur Appearance Themes

This folder owns the built-in Kangur appearance preset baselines and preset
composition helpers.

- `daily.ts`, `dawn.ts`, `sunset.ts`, `nightly.ts`: core preset baselines
- `others*.ts`: supplemental preset variants and effect packs
- `factory.ts`: factory/baseline preset composition helpers

These files are appearance-owned and should stay under `appearance/`, not at the
Kangur feature root.
