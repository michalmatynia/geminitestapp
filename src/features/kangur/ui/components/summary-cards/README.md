## Summary Cards

Shared summary-card and highlight-card primitives for Kangur UI.

- Keep reusable card/content primitives for progress, quest, icon-summary, labeled-value, and recommendation surfaces here.
- Keep folder-owned tests in [`__tests__`](./__tests__).
- Keep standalone widgets like `KangurPriorityAssignments` outside this folder unless they become generic card primitives.
