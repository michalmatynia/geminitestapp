# Kangur UI Profile Services

This folder owns profile-specific derived metrics and recommendation helpers
used by learner-profile and progress-facing UI surfaces.

## Layout

- `profile-mastery.ts`: mastery derivation helpers
- `profile-performance.ts`: performance summarization helpers
- `profile-recommendations.ts`: profile recommendation helpers
- `profile-streaks.ts`: streak calculations
- `profile-types.ts`: profile-service types
- `profile-utils.ts`: profile helper utilities
- `__tests__/`: profile-service tests for this folder
