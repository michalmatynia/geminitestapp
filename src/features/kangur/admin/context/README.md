# Kangur Admin Context

This folder owns admin-only React contexts and runtime providers used by the
Kangur authoring and editing workspaces.

## Layout

- `KangurQuestionIllustrationContext.tsx`: illustration and panel runtime context
- `KangurQuestionsManagerRuntimeContext.tsx`: questions-manager runtime provider
- `KangurTestQuestionEditorContext.tsx`: test-question editor context
- `LessonContentEditorContext.tsx`: lesson content editor context
- `LessonContentEditorRuntimeContext.tsx`: lesson content runtime provider and hook
- `LessonSvgQuickAddRuntimeContext.tsx`: SVG quick-add runtime context
- `__tests__/`: context/provider tests for this folder
